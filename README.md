# miracleheartlight-signal

サンリオピューロランドのパレードで使用されている「ミラクルハートライト」の音声信号を出力するためのWebアプリケーションです。

## Requirements

- Node.js

## Development

```bash
$ npm install
$ npm run dev
```

## Build

```bash
$ npm run build
```
